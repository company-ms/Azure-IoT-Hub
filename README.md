# Azure-IoT-Hub

!!Download lib:
Special library in https://github.com/andriyadi/AzureIoTHubMQTTClient !!

ESP control via iot hub:
-Remote firmware update  from server
-Emperature transmission with DHT11
-Remote restart

Note:
Token expires after ~ (10*24*3600) //seconds change in AzureIoTHubMQTTClient.h
Ater expiration ,a reboot is required



Instructions for setting IoT hub  https://docs.microsoft.com/ru-ru/azure/iot-hub/iot-hub-arduino-huzzah-esp8266-get-started
